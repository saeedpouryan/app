مستندات نرم‌افزار ماشین‌حساب با فریم‌ورک فلاتر



مقدمه

این پروژه یک نرم‌افزار ماشین‌حساب ساده است که با استفاده از فریم‌ورک Flutter و زبان Dart توسعه داده شده است. هدف این پروژه ارائه یک ابزار محاسباتی ساده و کارآمد برای کاربران است.


پیش‌نیازها

قبل از شروع به کار با این پروژه، نیاز است که پیش‌نیازهای زیر را فراهم کنید:

نصب Flutter SDK

نصب Dart SDK

نصب یک IDE مناسب مانند Android Studio یا Visual Studio Code


نصب و راه‌اندازی

برای نصب و راه‌اندازی پروژه، مراحل زیر را دنبال کنید:

پروژه را از مخزن نسخه‌کنترل (مثلاً GitHub) کلون کنید:

git@github.com:saeedpouryan/app.git


به دایرکتوری پروژه بروید:

cd repository


وابستگی‌های پروژه را نصب کنید:

flutter pub get


پروژه را اجرا کنید:

flutter run

راهنمای استفاده

برای استفاده از نرم‌افزار ماشین‌حساب، مراحل زیر را دنبال کنید: پس از اجرای برنامه، صفحه اصلی ماشین‌حساب نمایش داده می‌شود. با استفاده از دکمه‌های موجود می‌توانید عملیات‌های محاسباتی مختلفی را انجام دهید. نتیجه عملیات‌ها در نمایشگر بالای صفحه نمایش داده می‌شود.
